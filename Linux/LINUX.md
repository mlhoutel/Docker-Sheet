## Linux directory
```
/                                               *Root directory*  
├───bin                                         *Users programs*  
├───boot                                        *System booting*  
├───dev                                         *Devices (ex:CD)*  
├───etc                                         *Config folder*   
├───home                                        *Users folders*  
│     ├───userA  
│     └───userB  
├───lib                                         *Shared libraries*  
├───media                                       *Medias (ex:USB)*  
├───mnt                                         *Temporary medias*  
├───opt                                         *Programs add-ons*  
├───proc                                        *System infos*  
├───root                                        *Root user folder*  
├───sbin                                        *System programs*  
├───tmp                                         *Temporary folder*  
├───usr                                         *Users programs*  
└───var                                         *Datas and Logs*  
```
